Вариант №4
Задание 1
В заданиях на формирование матрицы предполагается, что размер ре- зультирующей матрицы не превосходит 10×10.
Даны целые положительные числа М. N и набор из N чисел. Сформировать матрицу размера М × N, у которой в каждой строке содержатся все числа из исходного набора (в том же порядке).
Задание 2
Дана матрица размера М × N. Поменять местами столбец с номером N и первый из столбцов, содержащих только отрицательные элементы. Если требуемых столбцов нет, то вывести матрицу без изменений.
